  <task id="13.3.1" title="Implement File Watcher System" est="M">
    Add `notify = "6.1"` to Cargo.toml.
    Requirement: SCRIPT-03.
    Create `ScriptWatcher` Resource.
    Watches `assets/scripts/` directory for write events.
    Emits a `ScriptChangedEvent` when a file is modified.
  </task>
  <task id="13.3.2" title="Implement Hot-Reload Logic" est="M">
    Implement `hot_reload_scripts_system`.
    Requirement: SCRIPT-03.
    Receives `ScriptChangedEvent`.
    Re-compiles the modified Lua file.
    Updates the handle in `ScriptRegistry`.
    Existing NPC entities will use the new handle on their next tick.
  </task>
  <task id="13.3.3" title="Implement Lua Error Reporting" est="S">
    Define `ScriptErrorEvent`.
    Requirement: SCRIPT-03.
    Capture Lua execution errors in `execute_npc_scripts_system`.
    Log errors with filename and line numbers to the engine console (or HUD chat).
  </task>
</tasks>
