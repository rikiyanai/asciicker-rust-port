`pub fn parse_world_section(data: &[u8]) -> Result<A3dWorld, AssetError>`:
1. Read first i32. If negative: format_version = -value, read next i32 as count. Else: format_version = 0, count = value.
2. For each instance: read mesh_id_len i32.
   - >= 0: mesh instance (read mesh_id string, inst_name with length prefix, f64x16 tm, flags, optionally story_id).
   - -1: sprite (read name, pos f32x3, yaw f32, anim i32, frame i32, reps i32x4, flags, optionally story_id).
   - -2: item (read proto_index i32, count i32, pos f32x3, yaw f32, flags, optionally story_id).
   - else: UnknownInstanceType error.
3. Apply .ply->.akm conversion for mesh_id.
4. story_id: read only if format_version > 0, else default to -1.

Helper functions: `read_i32`, `read_f32`, `read_f64`, `read_string(data, cursor, len)`, `read_len_prefixed_string(data, cursor)`. All use little-endian. Track cursor position throughout.

`pub fn parse_akm(text: &str) -> Result<AkmMesh, AssetError>`:
1. Validate "ply" header, "format ascii 1.0".
2. Parse element/property declarations. Build property_types vec for vertex properties: x=1, y=2, z=3, red/diffuse_red=4, green/diffuse_green=5, blue/diffuse_blue=6, alpha=7, else=0 (skip).
3. Read num_verts from "element vertex N", num_faces from "element face M".
4. After "end_header": read N vertex lines, extract values by property_types mapping.
5. Read M face lines: first token is vertex count. If negative, abs() it and mark freestyle=true. If count==2, parse as edge. If count==3, parse as face (3 indices + optional visual u32).
6. Default alpha=255 if not present.

Run tests -- MUST PASS.
