<tasks>
  <task id="9.3.1" title="Implement Proximity Pickup System" est="M">
    Implement `item_pickup_system`.
    Requirement: ITEM-03.
    When player triggers 'Interact', query `SpatialGrid` for nearest item entities.
    If within range, call `InsertItem` on player's inventory and despawn world entity.
    Update `SpatialGrid` to reflect removal.
  </task>
  <task id="9.3.2" title="Implement Item Drop System" est="M">
    Implement `item_drop_system`.
    Requirement: ITEM-03.
    Removes item from `Inventory` component.
    Spawns new item entity at player position with `ItemInstance` component.
    Registers entity in `SpatialGrid` and attaches to world scene.
  </task>
  <task id="9.3.3" title="Implement Consumable Use Logic" est="M">
    Implement `item_use_system`.
    Requirement: ITEM-03.
    Check `ItemProto::kind` for consumables ('F', 'D', 'P').
    Apply effects based on `sub_kind`: HP Potion → Heal, Food → Hunger restore.
    Remove item from inventory and call `DestroyItem` logic (no world spawn).
  </task>
</tasks>
