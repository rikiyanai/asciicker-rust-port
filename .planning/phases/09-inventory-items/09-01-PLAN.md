**Requirements**: ITEM-01

<objective>
Implement the Item Type Catalog (ItemProto) providing the data template for all items in the game.

Purpose: Defines the properties, visual representations (2D/3D sprites), and categories for every item type. This catalog is used by the inventory system for rendering and the equipment system for sprite selection.

Output: `src/game/items.rs` (ItemProto definitions + Library).
</objective>
