Create `golden.rs` with:
- `GoldenComparison` result struct:
  ```rust
  pub struct GoldenComparison {
      pub diff_count: usize,
      pub total_cells: usize,
      pub diff_percentage: f64,
      pub first_diffs: Vec<CellDiff>,  // first N diffs for diagnostics (cap at 20)
  }
  ```
- `CellDiff` struct:
  ```rust
  pub struct CellDiff {
      pub x: u32,
      pub y: u32,
      pub actual_gl: u16,
      pub actual_fg: [u8; 4],
      pub actual_bg: [u8; 4],
      pub expected_gl: u8,
      pub expected_fg: u8,  // xterm-256 palette index
      pub expected_bg: u8,  // xterm-256 palette index
  }
  ```
- `compare_ansi_grids` function:
  ```rust
  pub fn compare_ansi_grids(
      actual: &AsciiCellGrid,
      expected_gl: &[u8],
      expected_fg: &[u8],
      expected_bg: &[u8],
  ) -> GoldenComparison
  ```
  Implementation:
  - `total_cells = actual.cells_count()`
  - Assert expected arrays have same length as total_cells
  - Iterate all cells:
    - Get actual glyph (char_indices[i] as u8), actual fg, actual bg
    - Compare against expected_gl[i], expected_fg[i], expected_bg[i]
    - For fg/bg comparison: convert actual RGBA to xterm-256 palette index using the formula `16 + 36 * ((r + 25) / 51) + 6 * ((g + 25) / 51) + ((b + 25) / 51)` (matching C++ RGB2PAL)
    - If any of (gl, fg_idx, bg_idx) differ from expected: increment diff_count, record CellDiff if under cap
  - Compute diff_percentage = diff_count as f64 / total_cells as f64 * 100.0
  - Return GoldenComparison

- Helper `rgb_to_xterm256(rgba: &[u8; 4]) -> u8`:
  ```rust
  pub fn rgb_to_xterm256(rgba: &[u8; 4]) -> u8 {
      let r = ((rgba[0] as u16 + 25) / 51) as u8;
      let g = ((rgba[1] as u16 + 25) / 51) as u8;
      let b = ((rgba[2] as u16 + 25) / 51) as u8;
      16 + 36 * r + 6 * g + b
  }
  ```

- `assert_golden(comparison: &GoldenComparison, threshold_pct: f64)`:
  ```rust
  pub fn assert_golden(comparison: &GoldenComparison, threshold_pct: f64) {
      if comparison.diff_percentage > threshold_pct {
          let mut msg = format!(
              "Golden-file comparison FAILED: {:.2}% cells differ ({}/{}) -- threshold: {:.1}%\n",
              comparison.diff_percentage, comparison.diff_count, comparison.total_cells, threshold_pct
          );
          for diff in &comparison.first_diffs {
              msg.push_str(&format!(
                  "  ({},{}) gl: {} vs {}, fg: {} vs {}, bg: {} vs {}\n",
                  diff.x, diff.y,
                  diff.actual_gl, diff.expected_gl,
                  rgb_to_xterm256(&diff.actual_fg), diff.expected_fg,
                  rgb_to_xterm256(&diff.actual_bg), diff.expected_bg,
              ));
          }
          panic!("{}", msg);
      }
  }
  ```
