Create `camera.rs` with:
- `GameCamera` resource:
  - Input state: `pos: [f32; 3]`, `yaw: f32`, `zoom: f32`, `perspective: bool`, `scene_shift: [i32; 2]`
  - Derived state (recomputed each frame): `focal: f32`, `view_tm: [f64; 16]`, `inv_tm: [f64; 16]`, `view_dir: [f32; 3]`, `view_pos: [f32; 3]`, `view_ofs: [f32; 2]`, `frustum_planes: Vec<[f64; 4]>`
- `GameCamera::default()`: pos=[0,0,0], yaw=0, zoom=1.0, perspective=true, scene_shift=[0,0]
- `GameCamera::update(&mut self, dw: f64, dh: f64)` porting C++ render.cpp:2966-3034:
  - Compute yaw radians, sinyaw, cosyaw
  - scale = dh / dw.max(dh), zoom_scaled = zoom * scale, sin30 = 0.5
  - ds = 2.0 * zoom_scaled / VISUAL_CELLS
  - Build isometric view matrix: view_tm[0] = cosyaw*ds, view_tm[1] = -sinyaw*sin30*ds, etc.
  - Translation: view_tm[12] includes scene_shift[0]*2 (TRAP-R06), view_tm[13] includes scene_shift[1]*2
  - If perspective: focal = max(dw,dh)*2, view_dir = [-sinyaw, cosyaw, 0.0] (architectural, no vertical tilt), view_pos, view_ofs with scene_shift*2
- `GameCamera::extract_frustum_planes(&mut self, dw: f64, dh: f64)`:
  - Extract 4-6 frustum planes from the view-projection matrix
  - Each plane [a,b,c,d] normalized such that ax+by+cz+d >= 0 means inside
  - For isometric: planes from screen bounds transformed to world space
  - For perspective: planes from camera position + view direction + FOV

- `camera_input_system` Bevy system:
  - // TODO(Phase 6): Q/E will be replaced with physics-driven rotation via PhysicsIO.torque.
  - // These direct yaw modifications are TEMPORARY for Phase 5 camera navigation only.
  - Q/E rotation: `just_pressed(KeyQ)` -> yaw -= 45, `just_pressed(KeyE)` -> yaw += 45
  - WASD basic movement (speed=0.5)

- `camera_update_system` Bevy system:
  - Gets RenderConfig sample dimensions, calls camera.update() and camera.extract_frustum_planes()

Update `render/mod.rs`:
- Add `pub mod camera;`
- In CpuRasterizerPlugin::build():
  - `app.init_resource::<GameCamera>()`
  - Add camera systems to Update: `(camera_input_system, camera_update_system).chain()`

- `test_perspective_view_dir_horizontal`: view_dir[2] == 0 (no vertical tilt)
- `test_frustum_planes_count`: At least 4 planes extracted
