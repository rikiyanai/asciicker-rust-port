TDD: Write tests FIRST, then implement.

**state_machine.rs:** Port ActionState enum from game.h ACTION. Variants: None (idle/walk), Attack, Fall, Dead, Stand (standing up from fall). Implement `can_transition_to(&self, target) -> bool` with C++ guards:
- None: can always transition to
- Attack: blocked from Fall, Stand, Dead
- Fall: blocked from Dead
- Stand: only from Fall or Dead
- Dead: can transition to from any state
Method `try_transition(&mut self, target) -> bool`: checks guard, applies if valid, returns success.

**equipment.rs:** Port enums from game.h: Weapon(None, RegularSword, RegularCrossbow), Shield(None, RegularShield), Helmet(None, RegularHelmet), Armor(None, RegularArmor), Mount(None, Wolf, Bee), SpriteKind(Human, Wolf, Bee). All derive Clone, Copy, PartialEq, Eq, Default (#[default] on None variant).

SpriteReq component: kind, mount, action(ActionState), armor, helmet, shield, weapon fields. Method `sprite_index(&self) -> (SpriteKind, usize, usize, usize, usize)` returning 5D indices for array lookup. Guard equipment changes during attack (TRAP-G01): `can_change_equipment(&self) -> bool` returns false during Attack.

Mount physics size (TRAP-G02): `collision_dimensions(&self) -> (f32, f32)` returns (world_radius, world_height): Human=(2.0, 7.0), Wolf=(3.0, 9.0), Bee=(3.0, 9.0).

**animation.rs:** Constants from game.cpp: STAND_US_PER_FRAME(30_000), FALL_US_PER_FRAME(30_000), ATTACK_US_PER_FRAME(20_000), STEP_DIVISOR(1024), STEP_MASK(8*1024-1), STEP_OFFSET(3*1024).

AnimationState component: current_frame(u32), elapsed_us(u64), action_start_us(u64). Method `advance(action: ActionState, player_stp: i32, delta_us: u64) -> u32` returns frame index:
- None (walk): frame = ((player_stp + STEP_OFFSET) & STEP_MASK) / STEP_DIVISOR (8 frames)
- None (idle, player_stp == -1): frame = 0
- Attack: elapsed/ATTACK_US_PER_FRAME, returns frame count for attack anim
- Fall: elapsed/FALL_US_PER_FRAME
- Stand: elapsed/STAND_US_PER_FRAME
- Dead: last frame (frozen)

Tests: all transition guards (valid/invalid), equipment guard during attack, mount collision dimensions, animation frame advance for each action type, walk frame from player_stp.
